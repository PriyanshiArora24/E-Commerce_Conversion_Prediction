# E-Commerce Conversion Prediction

This project was developed as part of the **IIT Guwahati ML Bootcamp Hackathon**.

## Overview
Built a Logistic Regression model to predict whether a customer is likely to convert based on user behavior and demographic features.

## Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Workflow
- Data preprocessing
- Missing value handling
- Feature engineering
- Feature scaling
- Logistic Regression model training
- Model evaluation

## Results
- Test Accuracy: **72.7%**
- Evaluated using Precision, Recall, F1-score and ROC-AUC.

## Repository Contents
- `notebook.ipynb` – Complete implementation and analysis.
